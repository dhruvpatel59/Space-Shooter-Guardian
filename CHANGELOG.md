# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-10
### Added
- Initial release of Space Shooter Game.
- Core gameplay mechanics: movement, shooting, enemy spawning.
- Object pooling for bullets and enemies.
- Custom shader for ship effects.
- Complete documentation set for GitHub publication.
- CI/CD workflow for automated builds.
