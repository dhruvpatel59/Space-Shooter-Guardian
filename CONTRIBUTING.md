# Contributing to Space Shooter Game

First off, thank you for considering contributing to the Space Shooter Game! It's people like you that make it better.

## How Can I Contribute?

### Reporting Bugs
- Use the **Bug Report** issue template.
- Provide a clear, descriptive title.
- Explain the steps to reproduce the bug.
- Include screenshots or videos if possible.

### Suggesting Enhancements
- Use the **Feature Request** issue template.
- Explain why this feature would be useful.
- Describe how it should work.

### Pull Requests
- Follow the existing code style (SwiftLint conventions).
- Ensure all tests pass (if applicable).
- Keep PRs focused on a single change.
- Link the PR to a relevant issue.

## Development Setup
1. Fork the repo.
2. Clone your fork.
3. Open `Space Shooter Game.xcodeproj` in Xcode.
4. Make your changes in a new branch.

## Branch Naming Convention
- `feature/your-feature-name`
- `fix/bug-description`
- `docs/what-was-changed`

## Code of Conduct
By participating in this project, you agree to abide by the [Code of Conduct](./CODE_OF_CONDUCT.md).
